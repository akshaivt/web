LIST

HTML lists allow web developers to group a set of related items in lists.

ORDERED LIST

An ordered list in HTML is used to display a list of items in a specific, sequential order, where the order of the items is important. Ordered lists are defined using the ol tag, and each individual list item within the ordered list is defined using the li tag.

  1. Numbers
  2. Capital Letters
  3. Small letters
  4. Roman Numeral

UNORDERED LIST

An unordered list in HTML is used to present a collection of items where the order of the items is not significant. These lists are typically displayed with bullet points by default.

  • Disc
  • Circle
  • Square
  • None

DESCRIPTION LIST

A description list in HTML is used to present a list of terms and their corresponding descriptions or definitions. It is particularly useful for glossaries, FAQs, or any content where you need to associate a term with its explanation
